## 🛠️ 두 가지 제작 모델 가이드 및 비용 비교

### 🟢 1. 기본형 제작 파이프라인 (Gemini/GPT 스틸컷 이미지 연동)
* **제작 방식**: 
  1. 각 씬의 **Gemini / GPT 이미지 프롬프트**를 활용해 고품질 스틸 이미지 4컷을 생성합니다.
  2. 무료 동영상 편집 툴(예: CapCut)에 이미지를 올리고, 자연스러운 화면 디졸브/페이드아웃 효과와 줌인/줌아웃(Ken Burns effect) 모션을 적용합니다.
  3. ElevenLabs로 뽑은 성우 나레이션과 자막을 입혀 최종 렌더링합니다.
* **장점**: **제작 비용 0원**. LLM 및 이미지 생성 기본 크레딧 범위 내에서 빠르게 제작 가능합니다.

### 🔵 2. 고급형 제작 파이프라인 (힉스필드 등 동영상 API 연동)
* **제작 방식**:
  1. 각 씬의 **동영상 API 프롬프트**를 **힉스필드(Higgsfield)** 혹은 **런웨이 Gen-3** API에 호출하여 4초 길이의 역동적인 모션 클립 4개를 획득합니다.
  2. 비디오 클립 간의 트랜지션을 부드럽게 이은 후, 나레이션 오디오와 다이내믹 텍스트 캡션을 믹싱합니다.
* **장점**: 압도적인 몰입감과 전문적인 비디오 모션을 통해 투자사 피칭이나 공공기관 보고 시 제안의 신뢰도를 극대화합니다.
* **비용 최적화**: 힉스필드 API 호출 시, 초기 무료 테스트 크레딧을 우선 소진하고, 최종 본선 기획안에만 유료 렌더링을 집중 적용하여 비용 낭비를 예방합니다.
